## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [React](https://pt-br.reactjs.org/)
- [typescriptlang](https://www.typescriptlang.org/)

## 💻 Projeto

O projeto visa o estudo e desenvolvimento de uma aplicação em ReactJS. Com o intuito de expor os conhecimentos adquiridos ao longo dos estudos, a aplicação consiste em 2 ‘interfaces’ sendo uma sobre o autor e a outra a exposição de projetos desenvolvidos.

Para deixar a aplicação disponível utiliza-se o GitHub Pages como ferramenta de deploy.

## 📥 Instalação e execução

Faça um clone desse repositório

### Frontend

1. A partir da raiz do projeto, entre na pasta rodando `cd frontend`;
2. Execulte `npm install` para instalar as dependências;
3. Execulte `npm run start` para iniciar o servidor de desenvolvimento;
4. Abra `http://localhost:3000` para ver o projeto no navegador.
